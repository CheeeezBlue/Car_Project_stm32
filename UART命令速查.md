## 串口命令汇总

### 模式控制

| 指令    | 功能                                       | 说明                   |
| ------- | ------------------------------------------ | ---------------------- |
| `MD:1`  | 选择静止模式（禁止启动）                   | 安全模式，电机无输出   |
| `MD:2`  | 选择直线模式（航向串级 + 速度环）          | 选中后需按 KEY4 启动   |
| `MD:3`  | 选择循迹模式（灰度 PID + 速度环）          | 选中后需按 KEY4 启动   |
| `STOP`  | 停止运行 + 复位所有上层控制器              | 等同于 RunMode_Stop()  |
| `?`     | 查询模式/运行状态/航向/角速度/差速/轮速/PWM | —                      |
| `KEY1`~`KEY4` | 模拟物理按键                         | 效果等同于按键短按     |

### 速度控制

| 指令        | 功能                       | 参数范围   |
| ----------- | -------------------------- | ---------- |
| `SPD:50`    | 设置两轮速度%              | -100 ~ 100 |
| `L:50`      | 设置左轮速度%              | -100 ~ 100 |
| `R:50`      | 设置右轮速度%              | -100 ~ 100 |
| `L:50 R:30` | 分别设置左右轮             | -100 ~ 100 |
| `TGT:30`    | 设置脉冲目标 (pulses/10ms) | -135 ~ 135 |
| `O:80`      | 开环测试（直驱 PWM）       | -100 ~ 100 |

> 静止模式（MODE_STATIONARY）下禁止速度指令。直线/循迹模式下，非运行状态仍可设置速度和参数。

### 速度环 PID

| 指令     | 功能                     | 参数范围  |
| -------- | ------------------------ | --------- |
| `KP:500` | 速度环 Kp (/1000 → 0.5)  | 0 ~ 32767 |
| `KI:500` | 速度环 Ki (/1000 → 0.5)  | 0 ~ 32767 |
| `KD:70`  | 速度环 Kd (/1000 → 0.07) | 0 ~ 32767 |

### 前馈 + 死区 + 斜坡

| 指令       | 功能                         | 参数范围  |
| ---------- | ---------------------------- | --------- |
| `FFL:856`  | 左轮前馈系数 (/1000 → 0.856) | 0 ~ 32767 |
| `FFR:798`  | 右轮前馈系数 (/1000 → 0.798) | 0 ~ 32767 |
| `FFO:5000` | 死区偏移 (/1000 → 5.0)       | 0 ~ 32767 |
| `FDB:50`   | 前馈死区 (/1000 → 0.05)      | 0 ~ 32767 |
| `RMP:100`  | 目标斜坡步长 (/1000 → 0.1)   | 0 ~ 32767 |

### 直线模式 — 航向角 PID（外环）

| 指令       | 功能                            | 参数范围   |
| ---------- | ------------------------------- | ---------- |
| `YHP:1000` | 航向角 Kp (/1000 → 1.0)         | 0 ~ 32767  |
| `YHI:0`    | 航向角 Ki (/1000 → 0.0)         | 0 ~ 32767  |
| `YHD:0`    | 航向角 Kd (/1000 → 0.0)         | 0 ~ 32767  |
| `YRL:45`   | 目标角速度上限 (°/s)            | 0 ~ 360    |

> 航向角外环：`航向误差(°) → PID → 目标角速度(°/s)`，目标航向固定为 0°（启动时锁定当前朝向）。
> 仅在 MODE_STRAIGHT 下有效。

### 直线模式 — 角速度 PID（内环）

| 指令        | 功能                           | 参数范围   |
| ----------- | ------------------------------ | ---------- |
| `YAWP:1500` | 角速度 Kp (/1000 → 1.5)        | 0 ~ 32767  |
| `YAWI:200`  | 角速度 Ki (/1000 → 0.2)        | 0 ~ 32767  |
| `YAWD:10`   | 角速度 Kd (/1000 → 0.01)       | 0 ~ 32767  |
| `YAWM:5000` | 差速输出上限 (/1000 → 5.0)     | 0 ~ 32767  |

> 角速度内环：`目标角速度 - 实测角速度 → PID → 左右轮差速(pulses/10ms)`。
> 仅在 MODE_STRAIGHT 下有效。航向 PID 参数调好后一般不需要经常改动。

### 串级控制结构

```
航向目标(0°) - 航向角(积分所得)
         ↓ 外环 YHP/YHI/YHD
    目标角速度(≤YRL)
         ↓ 内环 YAWP/YAWI/YAWD
    左右轮差速(≤YAWM)
         ↓ CarControl 速度环
      左/右电机 PWM
```

### 巡线控制

| 指令        | 功能                           | 参数范围   |
| ----------- | ------------------------------ | ---------- |
| `LSPD:30`   | 巡线基础速度（±100）           | -100 ~ 100 |
| `LKP:3000`  | 巡线 Kp (/1000 → 3.0)          | 0 ~ 32767  |
| `LKI:100`   | 巡线 Ki (/1000 → 0.1)          | 0 ~ 32767  |
| `LKD:500`   | 巡线 Kd (/1000 → 0.5)          | 0 ~ 32767  |

> 仅在 MODE_LINE 下有效。

### 查询命令输出示例

```
STRAIGHT RUN HDG:-0.35 RT:2.1/1.8 DF:3.5 T:50,50 ENC:48,49 PWM:420,400
```

| 字段 | 含义 |
|------|------|
| STRAIGHT | 当前模式名 |
| RUN/STOP | 运行/停止状态 |
| HDG | 相对航向角 (°) |
| RT | 目标角速度(°/s) / 实测角速度(°/s) |
| DF | 左右轮差速 (pulses/10ms) |
| T | 左/右轮目标速度 |
| ENC | 左/右轮编码器脉冲 |
| PWM | 左/右电机 PWM 输出 |
