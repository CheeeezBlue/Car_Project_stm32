## 串口命令汇总

### 速度控制

| 指令        | 功能                       | 参数范围   |
| ----------- | -------------------------- | ---------- |
| `SPD:50`    | 设置两轮速度%              | -100 ~ 100 |
| `L:50`      | 设置左轮速度%              | -100 ~ 100 |
| `R:50`      | 设置右轮速度%              | -100 ~ 100 |
| `L:50 R:30` | 分别设置左右轮             | -100 ~ 100 |
| `TGT:30`    | 设置脉冲目标 (pulses/10ms) | -135 ~ 135 |
| `O:80`      | 开环测试（直驱 PWM）       | -100 ~ 100 |
| `STOP`      | 停止 + 复位 PID + 关 yaw   | —          |
| `?`         | 查询当前 TGT/ENC/PWM       | —          |

### 速度环 PID

| 指令     | 功能                     | 参数范围  |
| -------- | ------------------------ | --------- |
| `KP:500` | 速度环 Kp (/1000 → 0.5)  | 0 ~ 32767 |
| `KI:500` | 速度环 Ki (/1000 → 0.5)  | 0 ~ 32767 |
| `KD:70`  | 速度环 Kd (/1000 → 0.07) | 0 ~ 32767 |

### 前馈 + 死区

| 指令       | 功能                         | 参数范围  |
| ---------- | ---------------------------- | --------- |
| `FFL:856`  | 左轮前馈系数 (/1000 → 0.856) | 0 ~ 32767 |
| `FFR:798`  | 右轮前馈系数 (/1000 → 0.798) | 0 ~ 32767 |
| `FFO:5000` | 死区偏移 PWM% (/1000 → 5.0)  | 0 ~ 32767 |

### Yaw 环

| 指令        | 功能                           | 参数范围   |
| ----------- | ------------------------------ | ---------- |
| `YAW:30`    | 设置目标角速度 °/s（启用 yaw） | -360 ~ 360 |
| `YAW:0`     | 直行锁定（yaw 纠正）           | —          |
| `YAW:OFF`   | 关闭 yaw 环                    | —          |
| `YAWP:1500` | Yaw Kp (/1000 → 1.5)           | 0 ~ 32767  |
| `YAWI:200`  | Yaw Ki (/1000 → 0.2)           | 0 ~ 32767  |
| `YAWD:10`   | Yaw Kd (/1000 → 0.01)          | 0 ~ 32767  |
| `YAWM:5000` | Yaw 输出上限 (/1000 → 5.0)     | 0 ~ 32767  |

### 巡线控制

| 指令        | 功能                           | 参数范围   |
| ----------- | ------------------------------ | ---------- |
| `LINE`      | 启用巡线（或 `LINE:ON`）       | —          |
| `LINE:OFF`  | 关闭巡线 + 停车                | —          |
| `LSPD:30`   | 巡线基础速度（±100）           | -100 ~ 100 |
| `LKP:3000`  | 巡线 Kp (/1000 → 3.0)          | 0 ~ 32767  |
| `LKI:100`   | 巡线 Ki (/1000 → 0.1)          | 0 ~ 32767  |
| `LKD:500`   | 巡线 Kd (/1000 → 0.5)          | 0 ~ 32767  |
