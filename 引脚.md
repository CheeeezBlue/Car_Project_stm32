# STM32F103C8T6 电赛综合控制板引脚分配表

## 📋 总览

本控制板基于 STM32F103C8T6 (Cortex-M3, 48Pin LQFP) 设计，集成了双路电机驱动、编码器接口、蓝牙通信、灰度循迹、OLED显示等电赛常用功能模块。

---

## 🔌 各模块引脚详细分配

### 1. 双路编码器电机接口

| 电机 | 功能 | 引脚 | 定时器 | 备注 |
|------|------|------|--------|------|
| **左电机 (U1)** | 编码器 A 相 | **PB6** | TIM4_CH1 | 编码器模式输入 |
| | 编码器 B 相 | **PB7** | TIM4_CH2 | 编码器模式输入 |
| **右电机 (U2)** | 编码器 A 相 | **PA6** | TIM3_CH1 | 编码器模式输入 |
| | 编码器 B 相 | **PA7** | TIM3_CH2 | 编码器模式输入 |

---

### 2. TB6612 双路电机驱动模块

TB6612 控制引脚通过 P2/P3 插座引出，需根据实际接线确认对应 GPIO。

| 功能 | 插座引脚 | 典型连接 GPIO | 定时器 | 说明 |
|------|----------|---------------|--------|------|
| **左电机 PWM** | P2.PWMA | PA8 | TIM1_CH1 | 复用推挽输出 |
| **左电机方向** | P2.AIN1 | PB12 | - | 普通 GPIO 输出 |
| **左电机方向** | P2.AIN2 | PB13 | - | 普通 GPIO 输出 |
| **右电机 PWM** | P3.PWMB | PA9 | TIM1_CH2 | 复用推挽输出 |
| **右电机方向** | P3.BIN1 | PB14 | - | 普通 GPIO 输出 |
| **右电机方向** | P3.BIN2 | PB15 | - | 普通 GPIO 输出 |
| **待机控制** | P2.STBY | 固定高电平 | - | 低电平进入待机模式 |

---

### 3. 蓝牙通信模块 (USART1)

| 功能 | 引脚 | 复用功能 | 备注 |
|------|------|----------|------|
| **TX (发送)** | **PB10** | USART3_TX / I2C2_SCL | 图中标注 USART1，实际 USART1 默认在 PA9/PA10 |
| **RX (接收)** | **PB11** | USART3_RX / I2C2_SDA | 可能为 USART1 重映射或实际使用 USART3 |

> ⚠️ **引脚功能确认**：STM32F103C8T6 的 USART1 默认引脚为 PA9(TX)/PA10(RX)。图中标注 PB10/PB11 为蓝牙模块，如需使用 USART1，需确认是否配置为重映射模式（Remap），或实际使用 USART3。

---

### 4. OLED 显示屏 (I2C 接口)

| 功能 | 引脚 | 备注 |
|------|------|------|
| **SCL (时钟)** | **PB8** | I2C1_SCL / 软件模拟 I2C |
| **SDA (数据)** | **PB9** | I2C1_SDA / 软件模拟 I2C |
| **VCC** | 3.3V | - |
| **GND** | GND | - |

> **驱动方式**：支持硬件 I2C1 或软件模拟 I2C

---

### 5. 八路灰度循迹模块

| 通道 | 功能 | 引脚 | 输入模式 | 说明 |
|------|------|----------|------|------|
|  | **GND** | **GND** | - | - |
| **CN1** | **OUT** | **PB0** | 浮空输入模式 | 数据读取 |
|  | **AD0** | **PA5** | 推挽输出模式 | 通道选择 |
|  | **AD1** | **PA4** | 推挽输出模式 | 通道选择 |
|  | **AD2** | **PA3** | 推挽输出模式 | 通道选择 |
|  | **5V** | **电源模块 5V** | 浮空/上拉输入 | - |

---

### 6. 按键检测电路

| 按键 | 引脚 | 检测方式 | 电路特点 |
|------|------|----------|----------|
| **KEY1** | **PB3** | 低电平有效 | 10kΩ 上拉电阻，按键接地 |
| **KEY2** | **PB4** | 低电平有效 | 10kΩ 上拉电阻，按键接地 |
| **KEY3** | **PB5** | 低电平有效 | 10kΩ 上拉电阻，按键接地 |
| **KEY4** | **PC13** | 低电平有效 | 10kΩ 上拉电阻，按键接地 |

> **JTAG 注意**：PB3/PB4 为 JTAG 专用引脚（JTDO/TRACESWO、NJTRST）。若使用 SWD 调试且禁用 JTAG，可复用为普通 GPIO。

---

### 7. 蜂鸣器电路

| 功能 | 引脚 | 驱动方式 | 备注 |
|------|------|----------|------|
| **蜂鸣器控制** | **PB1** | 三极管驱动（S8050） | 高电平导通，低电平关闭 |

> 蜂鸣器独占 PB1 引脚，无冲突。

---

### 8. 电源输入/输出接口

| 接口 | 类型 | 电压 | 说明 |
|------|------|------|------|
| **CN5** | 外部输入 | 5.5V | 外部 5.5V 输入（兼容 5V） |
| **CN4** | 外部输入 | 3.3V | 外部 3.3V 输入 |
| **CN2** | 外部输入 | 12V | 电机驱动电源（VM） |

---

### 9. 排针引出 (J1/J2/H1/H2)

所有 GPIO 通过排针引出，方便扩展：

**J1/J2 (双排针)**：

- PC13-PC15
- PA0-PA12
- PA15
- PB0-PB1
- PB3-PB15
- 3.3V, 5V, GND

**H1/H2 (单排针，20Pin)**：

- H1: PB12, PB13, PB14, PB15, PA8, PA9, PA10, PA11, PA12, PA15,PB3, PB4,PB5,PB6, PB7,PB8,PB9,5V,GND,3.3V
- H2: GND,GND,3.3V,RESET, PB11, PB10, PB1, PB0, PA7, PA6, PA5, PA4, PA3, PA2, PA1, PA0, PC15, PC14, PC13, VBAT
