## 智能小车速度控制方案

### 一、核心策略
**带偏置前馈 + 速度斜坡规划 + 简易PID辅助**

### 二、核心公式

**1. 速度斜坡规划**

将用户输入的阶跃目标速度，转化为每个控制周期平滑递增/递减的规划速度：

$$Target_{ramp}[n] = \begin{cases} Target_{ramp}[n-1] + Step_{max}, & \text{if } Target_{user} > Target_{ramp}[n-1] + Step_{max} \\ Target_{ramp}[n-1] - Step_{max}, & \text{if } Target_{user} < Target_{ramp}[n-1] - Step_{max} \\ Target_{user}, & \text{otherwise} \end{cases}$$

- $Step_{max}$：每个控制周期允许的最大速度变化量，决定加减速快慢。

**2. 带偏置的前馈**

规划速度大于死区阈值时，根据稳态标定模型直接输出基础PWM：

$$PWM_{FF} = \begin{cases} k \cdot |Target_{ramp}| + b \cdot \text{sign}(Target_{ramp}), & |Target_{ramp}| > Deadband \\ 0, & |Target_{ramp}| \le Deadband \end{cases}$$

- $k$：线性增益系数（PWM/速度）
- $b$：死区偏置（电机刚好启动的最小PWM）
- $Deadband$：停车死区阈值，低于此值前馈置零

**3. PID闭环修正**

$$PWM_{PID} = K_p \cdot e(t) + K_i \cdot \int_0^t e(\tau) d\tau + K_d \cdot \frac{de(t)}{dt}$$

- 积分项做简单限幅：$I_{out} = \text{clamp}(I_{raw}, -I_{limit}, I_{limit})$

**4. 最终输出**

$$PWM_{out} = PWM_{FF} + PWM_{PID}$$

### 三、参考代码

```c
// ==================== 参数定义（左右轮独立，此处以单轮为例） ====================
#define MAX_SPEED_STEP   3.0f    // 斜坡步长：越大加减速越猛
#define DEADBAND_SPEED   1.5f    // 死区阈值：低于此速度前馈置零

// 稳态标定得到的模型参数
float k = 0.8f;     // 线性增益 (PWM / 速度单位)
float b = 18.0f;    // 死区偏置 (最小启动PWM)

// PID 参数
float Kp = 2.5f;
float Ki = 0.3f;
float Kd = 0.1f;
float I_limit = 50.0f;    // 积分限幅

// ==================== 全局变量 ====================
float target_ramp = 0.0f;           // 斜坡规划后的目标速度
float integral = 0.0f;              // 积分累加值
float prev_error = 0.0f;            // 上一次误差

// ==================== 1. 速度斜坡规划 ====================
float ramp_generator(float user_target) {
    if (user_target > target_ramp + MAX_SPEED_STEP)
        target_ramp += MAX_SPEED_STEP;
    else if (user_target < target_ramp - MAX_SPEED_STEP)
        target_ramp -= MAX_SPEED_STEP;
    else
        target_ramp = user_target;
    return target_ramp;
}

// ==================== 2. 带偏置前馈 ====================
float feedforward(float target) {
    if (fabsf(target) <= DEADBAND_SPEED)
        return 0.0f;
    return fabsf(target) * k + (target > 0 ? b : -b);
}

// ==================== 3. PID 计算 ====================
float pid_compute(float target, float actual, float dt) {
    float error = target - actual;
    
    // 积分累加 + 限幅
    integral += error * dt;
    if (integral > I_limit)  integral = I_limit;
    if (integral < -I_limit) integral = -I_limit;
    
    // 微分
    float derivative = (error - prev_error) / dt;
    prev_error = error;
    
    return Kp * error + Ki * integral + Kd * derivative;
}

// ==================== 4. 主控制循环（每周期调用一次） ====================
void speed_control_loop(float user_target, float encoder_speed, float dt) {
    // 第一步：速度斜坡规划
    float target = ramp_generator(user_target);
    
    // 第二步：计算前馈
    float ff_out = feedforward(target);
    
    // 第三步：计算PID
    float pid_out = pid_compute(target, encoder_speed, dt);
    
    // 第四步：叠加输出
    float pwm_out = ff_out + pid_out;
    
    // 输出限幅（根据硬件PWM范围）
    if (pwm_out > 100.0f)  pwm_out = 100.0f;
    if (pwm_out < -100.0f) pwm_out = -100.0f;
    
    set_motor_pwm(pwm_out);
}
```

### 四、标定说明

**步骤1**：关掉PID，给电机固定PWM（如 20, 30, 50, 70），测量各占空比下的稳态速度。  
**步骤2**：用直线 $PWM = k \cdot Speed + b$ 拟合数据，得到 $k$ 和 $b$。左右轮分别操作。  
**步骤3**：填入代码中的 `k` 和 `b`，开启PID，先调 `MAX_SPEED_STEP` 到加减速舒适，再微调 Kp/Ki/Kd。

- **全速段一致表现**：低速不卡顿，高速不超调
- **加减速平滑**：斜坡规划杜绝电流冲击
